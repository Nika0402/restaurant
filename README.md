# Restaurant

Site vitrine pour un restaurant, développé en HTML, CSS et JavaScript.

## Fonctionnalités
- Page d'accueil
- Menu du restaurant
- Commande en ligne
- Réservation de table
- Page de contact

## Technologies utilisées
- HTML5
- CSS3
- JavaScript

## Installation en local
1. Cloner le dépôt
```bash
   git clone https://github.com/Nika0402/restaurant.git
```
2. Ouvrir le fichier `index.html` dans un navigateur
